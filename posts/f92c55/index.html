2023年度总结

时光如梭，岁月无情! 2023年就快要过去了，小孩睡着了，静下心来回顾一下，2023年的自己是怎么样走过来的呢？

1.学习成长

  1. 深入阅读 Percon-Mongodb-Operator 源码，大概阅读 ElasticSearch 官方 Operator 源码；
  2. 浅读 Karmada 多集群管理源码，并初步测试了 Karmada 多集群管理，跨集群服务治理等功能；
  3. 深入阅读 Istio 源码，了解 Istio 整个注入过程以及流量过滤等；

2.休闲娱乐

大部分周末会带小孩在广州市内玩，但是因为小孩太小，没有出去太远的地方玩。希望明年能规划一下出去一家三口旅游，比如珠海的海岛上，看看清澈的海水。

3.工作事业

  1. 各中间件容器化项目推进，主要负责整体方案设计和项目管理；
  2. Istio 生产环境上线，主要承担 Istio 环境搭建维护，针对Envoy启动慢进行优化，Envoy启动从8分钟到10几秒产生质的变化；
  3. 随着公司上层成本压力，而 Istio 消耗资源实在太大，Karmada多集群管理方案预研；

4.家庭生活

时光如流水，一眨眼小孩从满地爬到现在满院跑，为人父母方知父母不易，每周尽量花时间带小孩出去周边玩，给他一个健康快乐的童年，尽量抽时间陪伴他，与他一起成长。 但是有些时候情绪难控，会大声吼他，这个也导致他总是黏着妈妈，有些时候我甚至都吃老婆的醋，为什么小孩总是黏着妈妈，有时明明是妈妈弄哭他也还是找妈妈抱,不要爸爸抱，也辛苦孩子妈妈每天在家办公还要照顾他。

5.身心健康

还算OK，整体大病没有，估计有点慢性支气管炎，总有一种痰咳不出的感觉，明年加强身体锻炼，跑步运动不低于200公里，提高自身免疫。

6.收支理财

没有晋升，薪资稍微涨了一丢丢，好过没有；没有做一些理财计划，有点小遗憾。

End：总结与期待

2023年整体比较平稳，工作没有太突出的业绩，个人成长在 Kubernetes 开源项目的知识技能有比较大的成长，阅读了一些开源项目源码，Kubernetes 源码也大概翻阅了一番，对一些核心组件有了更深一步的了解，夯实了自身技能。

给2024年立个FLAG:

[ ] 至少阅读一本非技术类书籍，暂定心理学方面

[ ] 至少阅读一本技术类书籍

[ ] 阅读亲子教育方面书籍

[ ] 完成200公里跑步运动

[ ] 完成一种中间件 Operator 开发

[ ] 能有一个自己的小 House
